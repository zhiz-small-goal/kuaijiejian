## 安装和启动

### 系统要求

- **操作系统**：Windows 10 或更高版本
- **运行环境**：.NET 8.0 Runtime（如未安装，首次运行会提示）
- **Photoshop 版本**：Adobe Photoshop CS6 或更高版本
- **磁盘空间**：至少 50MB 可用空间

### 安装步骤

1. **解压工具包**
   ```
   将压缩包解压到任意位置，推荐路径：
   C:\Program Files\PS快捷键工具\
   或
   D:\Tools\PS快捷键工具\
   ```

2. **验证文件完整性**
   
   确保以下文件和文件夹存在：
   - `kuaijiejian.exe` - 主程序
   - `HandyControl.dll` - UI 库
   - `PhotoshopScripts/` - 脚本文件夹（包含多个 .jsx 文件）
   - 其他 .dll 和 .json 配置文件

3. **首次启动**
   
   双击 `启动工具.bat` 或 `kuaijiejian.exe` 启动程序

### 启动方式

#### 方式一：使用启动脚本（推荐）
```batch
双击 "启动工具.bat"
```
优点：自动处理编码，显示友好的启动信息

#### 方式二：直接运行
```batch
双击 "kuaijiejian.exe"
```
优点：启动速度更快

#### 方式三：创建快捷方式
1. 右键点击 `kuaijiejian.exe`
2. 选择"发送到" → "桌面快捷方式"
3. 可以将快捷方式放到开始菜单或任务栏

## 基础功能

### 1. 添加快捷键

#### 操作步骤

1. **点击"添加功能"按钮**
   
   主窗口中的绿色"添加功能"按钮

2. **在弹出窗口中配置**
   
   ```
   ┌─────────────────────────────────────┐
   │  添加新功能                          │
   ├─────────────────────────────────────┤
   │  快捷键：[按下你想要的快捷键]        │
   │                                     │
   │  功能类型： [下拉选择]               │
   │   ├─ 新建图层                        │
   │   ├─ 保存文档                        │
   │   ├─ 路径转形状                      │
   │   └─ ...更多功能                     │
   │                                     │
   │  按钮名称： [输入显示名称]           │
   │                                     │
   │  按钮颜色： [选择颜色]               │
   │                                     │
   │        [确定]  [取消]                │
   └─────────────────────────────────────┘
   ```

3. **快捷键输入说明**
   
   - 点击"快捷键"输入框
   - 按下您想要的键盘组合
   - 支持的修饰键：`Ctrl`、`Shift`、`Alt`
   - 支持的主键：字母、数字、F1-F12等
   
   **示例：**
   - `Ctrl+Shift+N` - 新建图层
   - `Alt+S` - 快速保存
   - `F5` - 刷新操作
   - `Ctrl+Alt+P` - 路径转形状

4. **选择功能类型**
   
   从下拉菜单选择要执行的 Photoshop 操作

5. **输入按钮名称**
   
   给这个快捷键起个好记的名字，会显示在按钮上

6. **选择按钮颜色（可选）**
   
   可以为按钮设置不同颜色，方便区分

7. **点击确定保存**

#### 快捷键设计建议

```
✅ 推荐的快捷键组合：
- Ctrl+Shift+[字母]  （常用功能）
- Ctrl+Alt+[字母]    （次要功能）  
- Ctrl+[F1-F12]      （快速访问）
- Alt+[数字]         （数字编号功能）

❌ 避免使用：
- Ctrl+C, Ctrl+V 等系统快捷键
- Ctrl+S（可能与Photoshop冲突）
- 单个字母（会影响文字输入）
```

### 2. 使用快捷键

#### 方式一：键盘快捷键

1. 确保 Photoshop 正在运行
2. 在 Photoshop 中工作时，直接按下设置的快捷键
3. 工具会自动执行对应的操作

**注意事项：**
- 工具需要在后台运行（可以最小化）
- 快捷键在全局生效，不限于 Photoshop 窗口
- 如果快捷键没反应，检查工具是否正在运行

#### 方式二：点击按钮

1. 切换到工具窗口
2. 直接点击对应的功能按钮
3. 效果和按快捷键完全相同

### 3. 编辑和删除快捷键

#### 编辑快捷键

1. 找到要编辑的功能按钮
2. 点击按钮旁边的 **"..."** 或右键点击按钮
3. 选择 **"编辑"**
4. 在编辑窗口中修改设置
5. 点击确定保存

#### 删除单个快捷键

1. 点击按钮旁边的 **"..."** 或右键点击按钮
2. 选择 **"删除"**
3. 确认删除

#### 批量删除

1. 点击主窗口的 **"批量删除"** 按钮
2. 在列表中勾选要删除的功能
3. 点击 **"删除选中项"**
4. 确认删除

### 4. 更改按钮颜色

#### 单个按钮改色

1. 点击按钮旁边的 **"..."** 菜单
2. 选择 **"更改颜色"**
3. 在调色板中选择新颜色
4. 点击确定，颜色立即生效

#### 主题配色

1. 点击 **"主题设置"** 按钮
2. 选择预设主题或自定义颜色方案
3. 应用到所有按钮

### 5. 显示模式切换

工具支持三种显示模式：

#### 模式一：悬浮窗模式（TopMost）

```
特点：
✓ 窗口始终在最前面
✓ 不会被其他窗口遮挡
✓ 方便随时点击使用

适用场景：
- 需要频繁使用工具
- 双屏工作，工具放在副屏
- 初学者熟悉功能

切换方法：
设置 → 显示模式 → 悬浮窗模式
```

#### 模式二：桌面模式（Desktop）

```
特点：
✓ 普通窗口模式
✓ 可以被其他窗口遮挡
✓ 可以最小化到任务栏

适用场景：
- 偶尔使用工具
- 主要通过快捷键操作
- 需要更大的屏幕空间

切换方法：
设置 → 显示模式 → 桌面模式
```

#### 模式三：隐藏模式（Hidden）

```
特点：
✓ 完全隐藏窗口
✓ 仅保留快捷键功能
✓ 最大化屏幕空间

适用场景：
- 熟练用户
- 完全依赖快捷键
- 需要全屏工作

切换方法：
设置 → 显示模式 → 隐藏模式

恢复显示：
按快捷键 Ctrl+Alt+H
```

## 高级功能

### 1. 图层选择器

某些功能需要选择特定图层才能执行，这时会弹出图层选择器。

#### 使用场景

- 删除图层
- 复制图层
- 合并图层
- 栅格化图层

#### 操作步骤

1. **触发图层选择器**
   
   点击需要选择图层的功能按钮

2. **在弹出窗口中选择图层**
   
   ```
   ┌─────────────────────────────────────┐
   │  选择图层                            │
   ├─────────────────────────────────────┤
   │  [ ] 图层 1                          │
   │  [√] 图层 2  ← 选中                  │
   │  [ ] 背景                            │
   │  [ ] 文字图层                        │
   │                                     │
   │  [√] 记住选择，下次自动执行          │
   │                                     │
   │        [确定]  [取消]                │
   └─────────────────────────────────────┘
   ```

3. **勾选"记住选择"（可选）**
   
   勾选后，下次执行相同操作会自动使用上次的图层选择，不再弹窗

4. **点击确定执行**

### 2. Photoshop 动作集成

可以通过工具执行 Photoshop 中录制的动作（Action）。

#### 前置要求

1. 在 Photoshop 中已经录制好动作
2. 知道动作的名称和所属动作组名称

#### 添加动作快捷键

1. 点击 **"添加功能"**
2. 功能类型选择 **"执行动作"**
3. 输入动作信息：
   - **动作组名称**：动作所在的组（Set）
   - **动作名称**：具体动作的名称
4. 设置快捷键和按钮名称
5. 保存

#### 手动指定动作

如果不想添加到主界面，可以使用手动执行：

1. 点击 **"手动执行动作"** 按钮
2. 输入动作组名称和动作名称
3. 点击执行

工具提供多种路径转换功能：

```
功能说明：
将当前路径转换为矢量形状图层

```
```
功能说明：
```

```
使用条件：

应用场景：
```

### 4. 保存功能详解

工具提供多种保存方式，适应不同场景：

#### 4.1 普通保存

```
SaveDocument.jsx
- 相当于 Ctrl+S
- 保存当前文档
- 如果是新文档会提示另存为
```

#### 4.2 安全保存（SafeSave）

```
SafeSave_Fixed.jsx / SafeSave_Silent.jsx
- 保存前会检查文档状态
- 避免覆盖原文件的风险
- 适合重要文档
```

#### 4.3 智能保存（SmartSave）

```
SmartSave_Silent.jsx / SmartSave_V2.jsx
- 自动判断保存方式
- 处理各种特殊情况
- 最稳定的保存方案
```

#### 4.4 另存为 JPG

```
SaveAsJPG_Fixed.jsx / SaveAsJPG_Silent.jsx

选项：
- 质量设置（1-12）
- 文件大小控制
- 自动文件命名

应用场景：
- 导出网页用图
- 快速预览版本
- 批量输出
```

## 配置文件说明

工具的所有设置都保存在 JSON 配置文件中，可以手动编辑。

### 配置文件位置

```
工具目录/
├── functions_config.json      # 功能按钮配置
├── theme_config.json          # 主题颜色配置
└── display_mode_config.json   # 显示模式配置
```

### 1. functions_config.json

存储所有功能按钮的配置。

#### 文件结构

```json
[
  {
    "FunctionName": "新建图层",
    "HotkeyString": "Ctrl+Shift+N",
    "ScriptName": "CreateNewLayer.jsx",
    "Color": "#4A90E2",
    "ActionSetName": null,
    "ActionName": null
  },
  {
    "FunctionName": "执行动作示例",
    "HotkeyString": "Ctrl+Alt+A",
    "ScriptName": null,
    "Color": "#50C878",
    "ActionSetName": "我的动作组",
    "ActionName": "动作1"
  }
]
```

#### 字段说明

| 字段 | 说明 | 示例 |
|-----|------|------|
| `FunctionName` | 按钮显示名称 | `"新建图层"` |
| `HotkeyString` | 快捷键组合 | `"Ctrl+Shift+N"` |
| `ScriptName` | 要执行的脚本文件名 | `"CreateNewLayer.jsx"` |
| `Color` | 按钮颜色（十六进制） | `"#4A90E2"` |
| `ActionSetName` | 动作组名称（执行动作时使用） | `"我的动作组"` |
| `ActionName` | 动作名称（执行动作时使用） | `"动作1"` |

#### 手动编辑示例

**添加新功能：**

```json
{
  "FunctionName": "快速保存",
  "HotkeyString": "Alt+S",
  "ScriptName": "SmartSave_Silent.jsx",
  "Color": "#FF6B6B",
  "ActionSetName": null,
  "ActionName": null
}
```

**注意事项：**
- 编辑后需要重启工具才能生效
- 确保 JSON 格式正确（可用在线工具验证）
- 建议先备份原文件

### 2. theme_config.json

存储主题颜色配置。

```json
{
  "PrimaryColor": "#4A90E2",
  "SecondaryColor": "#50C878",
  "BackgroundColor": "#FFFFFF",
  "TextColor": "#333333"
}
```

### 3. display_mode_config.json

存储显示模式设置。

```json
{
  "DisplayMode": "TopMost",
  "WindowWidth": 800,
  "WindowHeight": 600,
  "WindowX": 100,
  "WindowY": 100
}
```

**DisplayMode 可选值：**
- `"TopMost"` - 悬浮窗模式
- `"Desktop"` - 桌面模式  
- `"Hidden"` - 隐藏模式

### 配置文件备份与恢复

#### 备份配置

```batch
# 手动备份
复制以下文件到安全位置：
- functions_config.json
- theme_config.json
- display_mode_config.json
```

#### 恢复配置

```batch
# 将备份的文件复制回工具目录
# 重启工具即可恢复配置
```

#### 重置配置

```batch
# 删除配置文件，工具会重新生成默认配置
1. 关闭工具
2. 删除所有 .json 配置文件
3. 重新启动工具
```

## 故障排除

### 常见问题

#### 1. 工具无法启动

**问题现象：**
- 双击无反应
- 闪退
- 提示缺少 .NET

**解决方案：**

```
步骤 1: 检查 .NET 环境
- 下载并安装 .NET 8.0 Runtime
- 下载地址：https://dotnet.microsoft.com/download/dotnet/8.0
- 选择 "Download .NET Runtime" → Windows x64

步骤 2: 检查文件完整性
- 确认所有 DLL 文件存在
- 特别是 HandyControl.dll

步骤 3: 以管理员身份运行
- 右键 kuaijiejian.exe
- 选择"以管理员身份运行"

步骤 4: 检查防火墙和杀毒软件
- 将工具添加到白名单
- 临时关闭杀毒软件测试
```

#### 2. 快捷键不响应

**问题现象：**
- 按快捷键没反应
- 点击按钮也不执行

**解决方案：**

```
检查清单：

□ Photoshop 是否正在运行
□ PhotoshopScripts 文件夹是否存在
□ 脚本文件（.jsx）是否完整
□ 是否有其他软件占用了相同快捷键
□ 工具是否在后台运行（检查任务栏）

具体操作：

1. 确认 Photoshop 运行：
   - 打开任务管理器
   - 查找 Photoshop.exe 进程

2. 确认脚本文件夹：
   - 打开工具目录
   - 确认 PhotoshopScripts 文件夹和 kuaijiejian.exe 在同一级
   - 进入文件夹，确认有多个 .jsx 文件

3. 测试基础功能：
   - 在 Photoshop 中打开一个文档
   - 点击工具的"新建图层"按钮
   - 如果点击有效但快捷键无效，说明是快捷键冲突

4. 解决快捷键冲突：
   - 换一个不常用的快捷键组合
   - 检查其他软件的快捷键设置
```

#### 3. 脚本执行出错

**问题现象：**
- Photoshop 弹出错误提示
- 操作没有完成
- 某些功能无法使用

**解决方案：**

```
常见错误和解决方法：

错误 1: "找不到路径"
原因：Photoshop 没有活动路径
解决：先用钢笔工具创建路径，再执行路径转换功能

错误 2: "没有可用的图层"
原因：当前文档没有图层或图层被锁定
解决：检查图层面板，确保有可编辑的图层

错误 3: "文档未保存"
原因：新建文档还没有保存过
解决：先手动另存为，再使用保存功能

错误 4: "动作不存在"
原因：指定的动作名称或组名称错误
解决：
  1. 打开 Photoshop 动作面板
  2. 确认动作的准确名称
  3. 注意大小写和空格

错误 5: "脚本错误"
原因：.jsx 脚本文件损坏或版本不兼容
解决：
  1. 重新解压工具包
  2. 确保使用完整的 PhotoshopScripts 文件夹
  3. 检查 Photoshop 版本（建议 CC 2019 或更高）
```

#### 4. 窗口显示异常

**问题现象：**
- 窗口显示不全
- 按钮重叠
- 界面错位

**解决方案：**

```
解决步骤：

1. 重置窗口大小：
   - 删除 display_mode_config.json
   - 重启工具

2. 检查屏幕缩放：
   - Windows 设置 → 显示 → 缩放与布局
   - 如果缩放不是 100%，可能导致显示问题
   - 调整为 100% 或设置高 DPI 兼容性

3. 设置高 DPI 兼容性：
   - 右键 kuaijiejian.exe → 属性
   - 兼容性选项卡
   - 勾选"替代高 DPI 缩放行为"
   - 选择"系统（增强）"

4. 更新显卡驱动：
   - 某些显示问题可能与显卡驱动有关
   - 访问显卡厂商网站下载最新驱动
```

#### 5. 配置文件损坏

**问题现象：**
- 工具无法保存设置
- 启动时提示配置错误
- 之前的设置消失

**解决方案：**

```
恢复步骤：

方案 1: 重置配置（无备份）
1. 关闭工具
2. 删除以下文件：
   - functions_config.json
   - theme_config.json
   - display_mode_config.json
3. 重新启动工具
4. 工具会创建新的默认配置

方案 2: 恢复备份（有备份）
1. 关闭工具
2. 用备份文件替换损坏的配置文件
3. 重新启动工具

方案 3: 手动修复
1. 用文本编辑器打开配置文件
2. 使用 JSON 验证工具检查错误
   推荐：https://jsonlint.com/
3. 修正 JSON 语法错误
4. 保存并重启工具
```

## 技术说明

### 工作原理

```
工具运行流程：

1. 用户按下快捷键
   ↓
2. 工具捕获快捷键事件
   ↓
3. 查找对应的功能配置
   ↓
4. 调用 PhotoshopHelper 执行脚本
   ↓
5. 通过 ExtendScript 与 Photoshop 通信
   ↓
6. Photoshop 执行 .jsx 脚本
   ↓
7. 返回执行结果
   ↓
8. 工具显示通知（如果有）
```

### 脚本文件说明

所有 .jsx 脚本都位于 `PhotoshopScripts/` 文件夹中。

#### 核心脚本

| 脚本文件 | 功能说明 | 使用场景 |
|---------|---------|---------|
| `CreateNewLayer.jsx` | 创建新图层 | 快速添加图层 |
| `DuplicateLayer.jsx` | 复制图层 | 备份图层内容 |
| `DeleteLayer.jsx` | 删除图层 | 清理无用图层 |
| `MergeLayer.jsx` | 合并图层 | 合并多个图层 |
| `ToggleLayerVisibility.jsx` | 切换图层可见性 | 显示/隐藏图层 |
| `RasterizeLayer.jsx` | 栅格化图层 | 转为像素图层 |

#### 文档操作脚本

| 脚本文件 | 功能说明 |
|---------|---------|
| `NewDocument.jsx` | 新建文档 |
| `SaveDocument.jsx` | 保存文档 |
| `SafeSave_Fixed.jsx` | 安全保存（稳定版） |
| `SmartSave_Silent.jsx` | 智能保存（静默） |
| `SaveAsJPG_Fixed.jsx` | 另存为 JPG |
| `FlattenImage.jsx` | 拼合图像 |

#### 路径操作脚本

| 脚本文件 | 功能说明 |
|---------|---------|
| `PathToShape_Simple.jsx` | 路径转形状（简单） |
| `PathToShape.jsx` | 路径转形状（完整） |
| `PathToMask_Silent.jsx` | 路径转蒙版 |
| `SetVectorMaskFeather.jsx` | 设置矢量蒙版羽化 |

#### 测试和诊断脚本

| 脚本文件 | 用途 |
|---------|-----|
| `TestSave.jsx` | 测试保存功能 |
| `DiagnoseSave.jsx` | 诊断保存问题 |
| `TestFileOperations.jsx` | 测试文件操作 |
