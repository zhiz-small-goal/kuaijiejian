## 🚀 首次发布步骤

### 1. 初始化 Git 仓库

```bash
# 在项目根目录执行
git init
git add .
git commit -m "feat: 初始提交 - Photoshop 快捷键工具 v1.0.0"
```

### 2. 创建 GitHub 仓库

1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 填写仓库信息：
   - **Repository name**: `photoshop-hotkey-tool`
   - **Description**: `一个强大的 Photoshop 快捷键增强工具，让您的设计工作更高效`
   - **Public/Private**: 选择公开或私有
   - **不要勾选** "Initialize with README"（我们已经有了）
4. 点击 "Create repository"

### 3. 推送代码到 GitHub

```bash
# 添加远程仓库（替换为您的用户名和仓库名）
git remote add origin https://github.com/yourusername/photoshop-hotkey-tool.git

# 设置默认分支名称
git branch -M main

# 推送代码
git push -u origin main
```

### 4. 创建首个 Release

1. 在 GitHub 仓库页面点击 "Releases"
2. 点击 "Create a new release"
3. 填写信息：
   - **Tag version**: `v1.0.0`
   - **Release title**: `v1.0.0 - 首次发布`
   - **Description**: 复制 CHANGELOG.md 中的内容
4. 上传发布文件：
   - 运行 `一键打包.bat` 生成发布包
   - 将生成的 zip 文件上传
5. 点击 "Publish release"
